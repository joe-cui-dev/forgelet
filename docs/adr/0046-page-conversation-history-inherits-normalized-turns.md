# Page Conversation History Inherits Normalized Turns

Each Page Conversation continuation receives the original browser capture plus the ordered user questions and normalized final answers from its ancestor Sessions. Streamed model text, tool activity, and other live presentation state do not enter Page Conversation History because they may be partial and are not authoritative outcomes. Generic Continuation Context remains compact evidence for other Workflows; Page Conversation History is the browser-specific conversational projection needed for references to earlier turns.

# Page Conversation Shapes Live in a Dependency-Free Leaf

Page Conversation's shared pure vocabulary — `PageConversationTurn`, `PageAnswer`, `PageAnswerGroundingStatus`, `PageBrief`, and `PageConversationAttemptKind` — belongs to the dependency-free `src/pageConversation` leaf. Browser extension modules consume it through type-only imports, so those imports erase before `buildExtension` copies its fixed compiled-file bundle; this keeps the ADR 0052 disposable projection compatible with the extension bundle boundary. This supersedes the local `learning.ts` Page Answer conversation re-declaration and the separate protocol `BrowserPageAnswer` mirror. The protocol owns the wire envelope rather than a duplicate payload type, and exact serialized completion-frame key tests are the explicit confirmation point for a wire-shape change, including reconsidering the ADR 0054 protocol version.

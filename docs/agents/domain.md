# Domain Docs

Forgelet is a single-context repository.

Before exploring an area:

1. Read the root `CONTEXT.md` for canonical domain language.
2. List `docs/adr/` and read only ADRs relevant to the area.
3. Treat current source and tests as implementation truth.

Use glossary terms in issue titles, specifications, tests, and design discussions. Surface conflicts with existing ADRs explicitly rather than silently overriding them.
